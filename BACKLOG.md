# SC2 Replay Analyzer - Backlog

## Tag System Enhancements

- **Interactive tag commands:**
  - `tag start` - Walk user through: enter tag name, choose date
  - `tag end` - Show available ongoing tags with start dates, let user choose
  - `tag remove` - Show all tags, let user choose which to remove
  - `tag edit` - Allow changing names or dates of existing tags
